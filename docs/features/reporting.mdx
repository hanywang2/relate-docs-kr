## Overview

Relate에서는 간단하지만 초기 B2B 영업을 운영하는데 중요한 세일즈 인사이트를 분석할 수 있습니다.

Relate에서는 모든 데이터를 전부 보여주는 것보다는 정말 필요한 데이터를 잘 보여주는 것이 중요하다고 생각하고 있습니다.

Relate Reporting & Analytics는 현재 2가지 데이터 인사이트를 보여줍니다.

1. Deal Conversion
2. Deal Performance

### Reporting 페이지 진입하기

현재 Reporting 페이지는 링크로만 진입이 가능합니다. 아래 링크를 복사/붙여넣기 하거나 [Reporting 페이지](https://app.relate.so/reporting)를 직접 방문하시면 됩니다.

```
https://app.relate.so/reporting
```


## Deal Conversion

Deal Conversion(전환율)은 파이프라인내 Deal 전체 숫자에서 특정 단계에서 다음 단계로 넘어갈 때, 혹은 Active -> Closed (Won/Lost) 상태로 변경이 되는 숫자를 계산합니다.

### Win Rate
세일즈 측정할 때 가장 중요한 것은 우리가 파이프라인내에 있는 모든 영업 기회 중에 실제로 몇 개를 수주에 성공하는가? 입니다. 이것을 Win Rate라고 부르는데요.

Win Rate를 알면 자연스럽게 파이프라인에 몇 개의 Deal을 추가해야 하는지도 알 수 있습니다.

![Win Rate](/docs/images/features/win-rate.png)

위 예시를 보면 총 8개의 Deal이 파이프라인에 있고, 그 중에 4개의 Deal을 성공시켰으니, Win rate는 50%가 됩니다.

이 데이터를 바탕으로 우리는 1개의 Deal을 수주하기 위해서는 2개의 Deal이 파이프라인에 있어야 함을 알 수 있습니다.

### Conversion Rate by Status

![Conversion Rate](/docs/images/features/conversion-rate.png)

파이프라인 단계별 전환율도 확인할 수 있습니다. 마우스를 그래프에 올리시면 전환율이 몇 퍼센트인지도 알 수 있습니다.

### Duration of each status

![Sales duration](/docs/images/features/duration.png)

파이프라인 각 단계에 Deal들이 평균적으로 며칠이나 머무는지 파악할 수 있습니다.

## Deal Performance
Deal Performance 페이지는 파이프라인 현황을 각종 그래프로 보여주는 기능입니다.

![Deal Performance](/docs/images/features/deal-performance.png)

### Total Deals

Total deals shows the number of active and closed deals in each status.

![Total deals](/docs/images/features/total-deals.png)

Total Deals 섹션은 현재 파이프라인에서 진행 중인 모든 Active 와 Closed (Won/Lost) Deal을 상태별로, 타입별로, 그리고 value period 별로 (one time, monthly, annual) 보여줍니다.

### Closed Deals

![Closed deals](/docs/images/features/closed-deals.png)

Closed deals는 파이프라인내 Won 과 Lost 상태인 deal 숫자의 트렌드를 보여줍니다.

### Newly Added Deals

![Newly added deals](/docs/images/features/new-deals.png)

Newly added deals는 파이프라인내 새롭게 추가된 Deal의 숫자를 시간대별로 보여줍니다.
