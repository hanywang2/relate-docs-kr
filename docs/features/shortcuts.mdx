<Tip>

**10x 빠르게 세일즈 하는 법**: Relate에는 더 빠르게 움직일 수 있도록 간단하고 기억하기 쉬운 단축키들을 지원하고 있습니다. **앞으로도 계속해서 더 많은 단축키를 지원할 예정입니다**.

</Tip>

### Inbox

| Action              | Shortcut                          |
| ------------------- | --------------------------------- |
| Inbox로 가기        | `⌘` + `1` (`Ctrl` + `1`)          |
| Archive로 가기      | `⌘` + `2` (`Ctrl` + `2`)          |
| Starred로 가기      | `⌘` + `3` (`Ctrl` + `3`)          |
| Archive 하기        | `E`                               |
| 아이템 Star(⭐)하기 | `S`                               |
| 아래로 가기 `↓`     | `J` or `↓`                        |
| 위로 가기`↑`        | `K` or `↑`                        |
| 댓글 등록하기       | `⌘` + `Return` (`Ctrl` + `Enter`) |

### 검색(Search) & Command Bar

Relate 앱내 어디서든 동작하는 유니버설 단축키로 현재는 Lead와 Contact 검색만을 지원하고 있지만, 추후 Relate 앱내 내비게이션과 Command Bar를 통한 신규 Lead, Contact, Deal 등록 등 다양한 기능을 지원할 예정입니다.

| Action                  | Shortcut  |
| ----------------------- | --------- |
| 검색 & Command Bar 열기 | `⌘` + `K` |
