## Import Template 다운로드 하기

[Admin 페이지](https://app.relate.so) 내에서 Import Data 메뉴를 클릭하신 뒤, 아래 스크린샷에 보이는 템플릿을 다운받으시면 됩니다.

Relate으로 고객 정보를 Import하실 때는 Organization과 People은 같이, Deal은 따로 Import하시면 됩니다. 템플릿이 나눠져 있으니 참고하시면 됩니다.

![Download leads](/docs/images/setup/download-organization-people-template.png)

## Import 준비하기

![Prepare imports](/docs/images/setup/prepare-imports.png)

다운받은 템플릿에 맞춰 기존에 관리하던 고객 정보를 입력해주시면 됩니다. 템플릿의 볼드 처리가 된 첫 두 줄은 그대로 유지한 채로 3번째 줄부터 추가하시면 됩니다.

템플릿의 첫 두 줄은 추가하시는 고객 정보를 Relate 내에서 어디에 추가할지 결정하는 역할을 합니다.

- 첫 번째 줄은 각 열에 (Column) 추가되는 정보가 Organization과 관련된 정보인지, People과 관련된 정보인지 결정합니다. 추가되는 정보가 기본적으로 제공하는 데이터 필드에 추가될지, 새로 만드는 커스텀 필드에 추가할지 결정합니다.

- 두 번째 줄은 각 열에 (Column) 추가되는 정보가 어떤 Organization/People 내에서 어떤 데이터 필드로 들어갈지 결정합니다.

### 예시:

예를 들어, A1은 organization/name으로, A 열에 추가되는 정보가 Organization과 관련된 정보이고, 또 기본으로 제공하는 데이터 필드인 Organization Name으로 추가된다는 것을 의미합니다.

그리고 A2의 Organization Name (required)는, A 열에 추가되는 정보가 Organization Name이라는 이름을 가진 데이터 필드로 들어가는 것을 의미합니다. 가로 안에 있는 (required)는 데이터 생성을 위해 항목이라는 것을 의미하고, Organization Name이 없는 데이터는 추가되지 않게 됩니다.

Assignee Email은 우리 팀 영업 담당자 중 한 명의 이메일을 추가하면 됩니다.

![Prepare imports](/docs/images/setup/import-assignee.png)

그리고 D1의 organization/custom_field/1은 D 열에 추가되는 정보가 Organization과 관련된 정보이고, 또 새롭게 추가한 커스텀 필드라는 것을 의미합니다.

D2의 \[Field Name\]은 새롭게 추가할 커스텀 필드의 이름입니다. 예를 들어 Organization에 추가하고 싶은 커스텀 필드가 "Industry" 라면 \[Field Name\]을 지우고 Industry로 바꿔주시면 됩니다. Industry를 입력하실 때는 대괄호 없이 입력하시면 됩니다.

![Prepare imports](/docs/images/setup/custom-field.png)

## Import 하기

Relate으로 Import하실 때는 .xlsx 파일이나 .csv 파일만 Import가 가능합니다.

준비하신 파일을 [Admin 페이지](https://app.relate.so) 내에서 Import Data 페이지로 들어가서, Import할 파일을 먼저 선택한 뒤 Import 버튼을 눌러주시면 됩니다.

![Import 하기](/docs/images/setup/import.png)

## 자주 묻는 질문 (FAQ)

<AccordionGroup>
	<Accordion title="Import 하기 전에 필요한 커스텀 필드를 미리 만들어야 하나요?">

미리 Admin Page에서 커스텀 필드를 만들지 않았어도 Import를 하게 되면 자동으로 커스텀 필드를 만들게 됩니다. 이럴 경우 자동으로 Text Format으로 Import됩니다.
	
만약 미리 만든 커스텀 필드로 데이터를 추가하고 싶은 경우 커스텀 필드의 이름을 동일하게 엑셀에서 입력해주셔야 합니다. (예: Inudstry로 커스텀 필드를 만든 경우 엑셀에서도 Industry를 사용해야 합니다. 만약 Industry Type으로 입력할 경우, 새로운 커스텀 필드를 생성하게 됩니다.)
</Accordion>

<Accordion title="Import 할 때 커스텀 필드의 데이터 포맷을 변경할 수 있나요?">

Import를 하면서 적용할 수 있는 데이터 포맷은 Text, Multi-line Text, Select, Resources: Users입니다. 기본적으로 Import를 하면 Text Format으로 생성됩니다.

만약 Import를 하는 데이터가 Text가 아닌 다른 유형의 데이터 포맷을 사용하고 싶은 경우, 미리 Admin Page에서 커스텀 필드를 (Text, Multi-line Text, Select, Resources: Users)로 만든 뒤 Import해야 합니다.

Resources: Users의 경우 팀원들의 이름이 아닌 이메일을 입력해야 데이터가 등록됩니다.
</Accordion>
</AccordionGroup>
