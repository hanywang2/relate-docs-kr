## Overview

Relate에 갖고 계신 기존 고객데이터를 쉽고 간편하게 import 할 수 있는 방법을 소개합니다.

<Check>

**1-point guide*

[Admin 페이지](https://app.relate.so/) 왼쪽 메뉴에서 Import Data를 클릭한 뒤, Import Organizations 또는 Import Deals 버튼을 클릭합니다.

![Relate Data](/docs/images/setup/import_menu1.png)

Relate에 고객 정보를 Import 할 때는 Organization과 People은 Import Organizations에서, Deal은 Import Deals에서 할 수 있습니다.

</Check>

--

## 1. Relate에 고객 정보 추가하기 전 꼭 알아야 할 것

### Organization, Person, Deal

Relate의 고객 정보는 크게 3가지로 나눠집니다. 
- [Organization (회사)](https://www.relate.kr/docs/features/organizations)
- [Person (회사 소속 이해관계자)](https://www.relate.kr/docs/features/people)
- [Deal (영업 거래)](https://www.relate.kr/docs/features/deal-pipeline)

![Relate Data](/docs/images/setup/org_person_deal.png)

<Info>

Relate 내에서 고객 정보를 분류할 때 Oragnization이 가장 기본이 되며, Person과 Deal은 Organization내에 속하게 되어 있습니다. 그렇기 때문에 Organization이 없이 Person과 Deal은 존재할 수 없습니다.

</Info>

### 필수 입력 정보 및 디폴트 데이터 필드

Organization, Person, Deal의 Default Data Field는 다음과 같습니다. 

Organization:
- **Organization Name (필수 입력 정보)**
- Website
- Description
- Assignee (팀 내 고객 담당자)

Person:
- **Person Name (필수 입력 정보)**
- Title
- Email
- Phone

Deal:
- **Organization (필수 입력 정보)**
- Person
- Value
- Prospect Status (잠재 고객 상태)
- Pipeline (2개 이상의 파이프라인을 사용할 경우만 해당)
- Pipeline Stage
- Assignee (팀 내 거래 담당자)
- Notes

### Default Data Field 이외의 다른 정보를 Import 하는 방법

Default Data Field 이외의 다른 데이터를 추가하기 위해서는 Import 하기 전에 먼저 [Custom Field](https://www.relate.kr/docs/features/custom-fields)를 만들어야 합니다. 

Custom Field를 Organization, Person, Deal에서 추가한 이후 Import 하면 됩니다.

![Relate Data](/docs/images/setup/custom_field_menu.png)

## 2. 고객 데이터 Import 하기

[Admin 페이지](https://app.relate.so/) 왼쪽 메뉴에서 Import Data를 클릭한 뒤, Import Organizations 또는 Import Deals 버튼을 클릭합니다.

![Relate Data](/docs/images/setup/import_menu1.png)

Relate에 고객 정보를 Import 할 때는 Organization과 People은 Import Organizations에서, Deal은 Import Deals에서 할 수 있습니다.

### Import 메뉴

![Relate Data](/docs/images/setup/import_menu2.png)

Import Oragnizations 또는 Import Deals 버튼을 누르면 위와 같은 화면으로 이동합니다. 

샘플은 오른쪽 상단의 Download example을 누르면 다운로드 받을 수 있습니다. 

이미 Import 할 파일이 준비되었다면 오른쪽 하단의 Next Step을 누르면 됩니다.

![Relate Data](/docs/images/setup/import_menu3.png)

다음 화면에서 가운데 있는 버튼을 누르거나 Import 할 파일을 드래그해서 추가합니다.

![Relate Data](/docs/images/setup/import_menu4.png)

파일이 잘 추가되었다면 하단의 Next Step을 눌러 다음 화면으로 이동합니다. 

![Relate Data](/docs/images/setup/import_menu5.png)

다음 화면에서 필요하다면 CSV 또는 XLSX 파일의 컬럼과 Relate의 데이터 필드를 매칭하면 됩니다. 매칭이 끝난 뒤 오른쪽 하단의 Next Step을 눌러 다음 화면으로 이동합니다. 

만약 Import 하는 과정에서 문제가 있는 부분은 빨간색으로 표시됩니다.

![Relate Data](/docs/images/setup/import_menu6.png)

Import가 성공적으로 끝나면 아래의 화면이 나옵니다. 

![Relate Data](/docs/images/setup/import_menu7.png)
