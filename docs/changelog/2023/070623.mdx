### 사용자 지정 노트 템플릿  

이제 나만의 노트 템플릿을 만들거나 기존 영업 방법론에 따라 미리 구축된 템플릿 중에서 선택할 수 있습니다. 이러한 템플릿은 영업 회의 노트를 체계적으로 정리할 수 있는 구조를 제공하여 중요한 정보를 일관성 있게 캡처할 수 있도록 도와줍니다.  

### 기본 제공 템플릿  

나만의 템플릿을 만들거나 미리 빌드된 템플릿 중 하나를 사용할 수 있습니다:  
- BANT  
- MEDDIC
- Relate 팀에서 사용하는 프레임워크

## Improvements & Fixes  
- 개선: Zapier 3.7.0 - 거래 찾기/생성/업데이트 작업이 추가되었으며, 이제 레코드 생성 또는 수정에 실패할 때 Zapier에 자세한 오류 메시지가 표시됩니다.  
- 개선: 이제 이메일에 스레드 소유자(받는 사람, 참조, 숨은 참조)가 없는 경우 릴레이트에서 답장/전달을 허용하지 않습니다.  
- 개선: 조직 세부 정보 페이지의 조직 웹사이트 URL 섹션을 제자리에서 편집할 수 있도록 변경했습니다.  
- 수정: 연락처로 등록되지 않은 참조 수신자가 "모두 답장"에서 누락되던 문제를 수정했습니다. 
- 수정: 참조한 이메일에 답장할 때 참조 받는 사람이 제대로 설정되지 않던 문제를 수정했습니다.  
- 수정: 초안 자동 저장 중 이메일을 보낼 수 없던 문제를 수정했습니다.  
- 수정: 받은 편지함 페이지에서 이메일 템플릿을 작성/편집/사용할 수 없던 문제를 수정했습니다.  
- 수정: 데이터 보기에 저장할 때 거래 수 필터가 초기화되던 문제를 수정했습니다.
