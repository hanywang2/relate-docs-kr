## Overview
오늘 업데이트에서는 다음과 같은 개선과 기능 추가가 있었습니다.
- Task Management 기능 추가
- [필수] 신규 Relate 앱 버전 설치
- 이메일 첨부 파일 다운로드 기능 추가
- Zapier 기능 개선

## Task Management

![task](/docs/images/changelog/task.png)

이제 Relate에서 영업과 고객 이슈 대응에서 활용할 수 있는 Task 관리가 추가되었습니다.

Relate의 Task v1.0은 Organization Detail Page (각 고객사 페이지)에서 관리합니다. 위 스크린샷에서 보이는 것처럼, 왼쪽 하단에서 Task 리스트를 확인하고 관리할 수 있습니다.

기본적인 기능은 간단하게 task 목록을 만들고, 완료 여부를 확인하고, 워크스페이스 유저(팀 구성원)를 assign 하실 수 있습니다.

## [필수] 신규 Relate 앱 버전 설치

**이제 Relate 신규 업데이트시 앱을 번거롭게 새로 설치하실 필요가 없게 되었습니다.** 앞으로는 자동으로 신규 업데이트가 진행이 될 예정이며, 앱을 실행하시기만 하면 됩니다.

단, 마지막으로 재설치를 해주셔야 Relate을 사용하실 수 있습니다.

재설치 방법은 다음과 같습니다.

1. 현재 사용하고 있는 레거시 Relate 버전을 언인스톨(uninstall)
2. [이 링크](https://download.relate.so/)에서 새로운 Relate 버전 설치
3. 이후로는 추가 설치 없이 자동으로 업데이트가 됩니다.

<Warning>현재 사용하고 있는 레거시 Relate 버전을 삭제하지 않으시면, 두개의 Relate 버전이 데스크톱에 남아 있을 수 있습니다.</Warning>

<Warning>현재 사용하고 계신 버전이 **0.14.0**이 아니라면, **0.14.0**으로 먼저 업데이트를 하신 다음에 위 프로세스를 따라서 재설치를 해주셔야 합니다.</Warning>

## 이메일 첨부 파일 다운로드
![file-attachment](/docs/images/changelog/file-attachment.png)
이제 Relate에 들어오는 이메일의 첨부파일을 다운로드 하실 수 있습니다. 이메일에 첨부파일이 존재하는 경우 위 스크린샷처럼 보이게 됩니다. 다운로드 버튼을 누르시면 로컬에 파일을 다운로드하실 수 있습니다.

## Fixes and Improvements
- Improved: Zapier에서 Organization Assignee와 Status 변경시 동작하는 Trigger 추가
- Fixed: Filter 관련 동작 오류 버그 수정
- Fixed: 이메일 포워드 할 때 Send 버튼이 Disabled 되어 있는 버그 수정
- Fixed: Organization Status 변경 후 Organization Detail Page 수정시 하단에 현재 정보가 보이는 버그 수정
- Fixed: Tilebox 이메일 연동 이슈 해결
- Fixed: Deal Pipeline 페이지 커스텀 필드로 필터할 때 발생하는 버그 수정
- Fixed: Zapier webhook trigger시 empty 값으로 보내지는 버그 수정
- Fixed: Organization Page에서 커스텀 필드로 정렬하면 아무것도 보이지 않는 버그 수정
