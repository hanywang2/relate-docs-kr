## Overview
Relate이 새로운 옷을 입었습니다. 👕 이번 업데이트에는 앱내 전반적인 UI 변경이 있었고, 메이저 버그 픽스들을 진행했습니다. 이번 체인지로그에는 주요 버그 및 UX 개선 사항들만 포함했습니다.

## UI Update

![ui-changes2](/docs/images/changelog/ui-changes2.png)

아직 앱내 몇몇 부분에서 기존 UI가 보이실 겁니다. 나머지 변경되지 않은 UI도 빠른 시일내에 새로운 UI로 업데이트 될 예정입니다.

## Fixes & Improvements

- Improved: Organization list에 `last_contacted_at` 칼럼 추가
- Improved: Org detail 에서 이메일 주소 텍스트 셀렉트 가능하게 개선
- Improved: 로그인시 로그인 로딩 상태 표시 처리
- Improved: 이메일내에 삽입된 이미지 및 첨부 이미지 디스플레이
- Improved: 이메일 템플릿에서 템플릿 이름을 적지 않으면 나오는 에러 메시지 구현
- Fixed: Deal list 정렬 이슈 수정
- Fixed:Deal list view에서 update하면 deal이 뷰에서 사라지는 이슈 수정
- Fixed: Org/People list datagrid 에서 발생한 여러가지 버그 수정
- Fixed: 이메일 스레드에 답장하거나 이메일 템플릿을 사용하면 기존에 작성 중이던 드래프트가 사라지는 이슈 해결
- Fixed: `Sent via Relate` signature가 이메일 템플릿을 사용할 때 두번 들어가는 이슈 수정
- Fixed: Datetime/Date 및 타임존 이슈 일원화 및 해결
- Fixed: 이메일 스레드가 정상적으로 접히지 않는 이슈 해결
