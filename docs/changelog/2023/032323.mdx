## Overview

오늘 업데이트에서는 다음과 같은 개선과 기능 추가가 있었습니다.

-   Relate 앱 로딩 속도를 크게 개선했습니다. 🏎️
-   앱 로딩 중임을 알 수 있도록 Inbox 옆에 로딩 스피너를 추가했습니다. ꩜
-   컨택을 .csv 파일로 익스포트할 수 있습니다.
-   “Status Updated At” 필터/정렬 항목을 추가했습니다.
-   Organization/People 리스트 UX를 개선했습니다.
-   이메일 주소/웹사이트url이 등록되어 있음에도 Organization 로고가 표시되지 않는 버그를 수정했습니다.

### 앱 로딩 속도 개선

![load-faster](/docs/images/changelog/load-faster.gif)

Relate 앱이 더 빠르게 로딩되도록 속도를 개선했습니다. 이제 전보다 훨씬 더 빨리 앱을 실행하실 수 있게 되었습니다.

![spinner](/docs/images/changelog/spinner.png)

앱 로딩 중임을 알 수 있도록 Inbox 옆에 로딩 스피너를 추가했습니다.

### CSV export button

![export-csv](/docs/images/changelog/export-csv.png)

이제 Relate 팀에게 별도로 요청하실 필요 없이 필요하실 때마다 컨택을 .csv 파일로 익스포트할 수 있습니다.

### Status Updated At Sort/Filter

![status-updated-at](/docs/images/changelog/status-updated-at.png)

`Status Updated At` 필터/정렬 항목을 통해 Organization Status가 변경되는 순으로 정렬하실 수 있습니다.
