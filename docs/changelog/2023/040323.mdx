## Overview

빠르게 움직이는 스타트업의 B2B 영업, 고객성공 팀에서는 (거의) 동일한 내용의 이메일을 고객에게 보내는 경우가 많습니다. 

세일즈 아웃바운드/콜드메일, 팔로업 메일, 제안 메일 등의 이메일을 거의 동일한 포맷을 기반으로 수신자의 맥락과 상황에 맞게끔 개인화를 해서 보내는데요.

세일즈와 고객 팀이 이런 작업을 쉽고 간편하게 할 수 있도록 이메일 템플릿 기능을 추가했습니다.

이메일 템플릿을 사용하기 위해서 먼저 Organization Detail Page로 들어갑니다. 그 다음, 신규 이메일을 작성하거나 답장하기 기능을 통해 이메일 Draft 창에서 `Use template` 버튼을 누릅니다. Template 창에서 기존 템플릿들도 수정/삭제 할 수 있고, 새로운 템플릿을 만들 수도 있습니다.

![email-template2](/docs/images/changelog/email-template-2.png)

**모든 템플릿은 워크스페이스내 다른 사용자들에게도 공유가 됩니다.**

얼리 액세스 고객은 현재 무제한으로 이메일 템플릿을 만들어서 사용하고 팀원 분들과 공유할 수 있습니다.

## Inbox 알림 방식 개편

![inbox-overhaul](/docs/images/changelog/inbox-overhaul.png)

Inbox에서의 알림 방식을 개편 했습니다. 이제 Inbox에 들어오는 모든 업데이트 사항은 Organization 단위로 들어오게 됩니다.

Inbox에 들어오는 업데이트 내역들:
* 이메일이 새로 들어오거나 기존 스레드에 이메일이 추가되었을 때,
* 신규 노트가 생성되었을 때,
* 노트/이메일 등에 댓글이 달렸을 때,
* 그리고 Deal 내역이 추가/업데이트 되었을 때 (현재는 '알림'만 오고 Inbox에서 해당 Deal 자체가 보이지는 않습니다.)
