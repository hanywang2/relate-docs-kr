안녕하세요,

Relate 팀 공동창업자 Chris 입니다.

## Zapier 연동 1.0
Zapier 연동의 첫 버전을 출시했습니다. 이제 스티비, 메일침프, Typeform, 자체 웹사이트 등에서 들어오는 Lead와 Contact 정보를 Relate 에 자동으로 생성할 수 있습니다.

Relate 팀은 Typeform 을 통해서 Lead 정보를 받고 있는데요. Zapier 1.0 연동으로 이제 Typeform 에 들어온 Lead 정보를 수기로 옮기지 않고, 자동으로 입력할 수 있게 되었습니다.

Zapier 연동이 필요하신 경우, 저희 팀에게 메시지 주시면 셋팅과 워크플로우 설계를 도와드리겠습니다 (직접 셋팅하실 수도 있습니다).

[Zapier 란?](https://brunch.co.kr/@seoup5024/80)

다음 Zapier 에는 노트 생성, 딜(Deal) 생성, 슬랙(Slack), 잔디(Jandi), 인터콤(Intercom)과 같은 툴과의 연동 및 자동화를 구현할 예정입니다.

## 로딩 속도 빠르게 개선
그동안 이 로켓 로딩화면이 없어지려면 꽤 오랜 시간이 걸렸는데요. 이제 평균 4초대로 로딩 속도를 개선하였습니다.

언제나 뭐가 잘 안된다면, Ctrl+Shift+R (맥: Cmd+Shift+R)을 누르시면 새로고침하실 수 있습니다.

## Import 개선
이제 Relate 에 엑셀 파일을 import 할 때, custom field 에도 import 할 수 있습니다. 그 뿐 아니라 Deal Pipeline 에 들어갈 Deal 정보도 엑셀 파일 import 을 통해 생성하실 수 있습니다.

Import가 필요하실 때 저희에게 메시지 주시면 바로 import 를 도와드리겠습니다.

## 곧 추가될 것들
- Lead/Contact Filtering/Sorting: Lead 와 Contact 리스트에서 필요한 정보만을 볼 수 있도록 적절한 Filtering 과 Sorting 을 적용할 수 있게 됩니다.
- 딜 파이프라인 Filtering/Sorting: 파이프라인에서 필요한 정보만을 볼 수 있도록 적절한 Filtering 과 Sorting 을 적용할 수 있게 됩니다.
- 안정성 개선: Relate 프로덕트의 전반적인 속도 개선과 안정성 개선이 이루어질 예정입니다.
