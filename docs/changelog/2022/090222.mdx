Relate의 전반적인 UI 업데이트가 있었습니다. 가장 눈에 띄는 업데이트는 Contact의 위치 이동입니다. 또한, 왼쪽 내비게이션에 사용되는 아이콘을 변경했습니다.

Contact을 따로 분리한 Contact Detail Page가 새로 생겼습니다. Contact 페이지 진입은 Contact에서 바로 클릭하셔서 들어가거나, Lead Detail Page에서 Contact 카드를 클릭해서 진입할 수 있습니다.

추가로, 아래 컴포넌트들의 UI가 변경되었습니다.

Deals 페이지와 Lead/Contact 페이지에서 보이는 Deal 카드 UI 개선

검색 Bar 리사이징 및 왼쪽으로 위치 이동

Empty State 아이콘 업데이트

업데이트 사항을 사용해보시려면 [Relate 버전을 0.10.0](https://app.relate.so/downloads)으로 업데이트하시면 됩니다.

## Fixes & Improvements
- 왼쪽 내비게이션에서 Deals, Contacts, Leads 순서를 변경했습니다.
- 왼쪽 내비게이션에서 Views 를 모아서 볼 수 있는 탭을 새로 생성했습니다.
- 이제 Relate 화면 윈도우 사이즈를 기억해서 다시 Relate을 실행하실 때 동일한 사이즈로 보여줍니다.
- 딥링크가 지원됩니다. `relate://#/archive`를 브라우저 주소창에 입력해보세요! Relate의 아카이브 탭이 열립니다.
- (구조적으로만) Lead가 없는 Contact도 생성이 가능하도록 변경되었습니다.
- 왼쪽 내비게이션에 스크롤을 추가했습니다.
- Contact이 저장이 되지만, 저장되지 않은 것처럼 보이는 버그를 수정했습니다.
